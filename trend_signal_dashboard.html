데이터 연결 설정

팀원이 입력하는 Google Form → 응답 시트를 웹에 게시(CSV) 한 뒤, 그 CSV 링크를 아래에 붙여넣고 코드 상단 SHEET_CSV_URL 값을 교체해 GitHub Pages에 재배포하세요. 링크를 비워두면 데모 데이터로 동작합니다.

※ 컬럼명은 아래 표의 이름 또는 괄호 안 유사 표현(띄어쓰기 차이 등) 중 하나면 인식됩니다. O/X, Y/N, 예/아니오는 자동으로 같은 값으로 처리됩니다.
